
## Zettelkasten笔记系统的搭建
卡片笔记系统：
1、创建四个文件夹：files、referencenotes、templates、zk
2、修改新笔记创建的默认位置为：zk
3、修改附件的默认存放位置
4、选择你喜欢的界面主题

obsidian链接用法：
链接到某一篇笔记：``[[ ]]``

链接到某一篇笔记中的某个标题：``[[ # ]]``

链接到某一篇笔记中的某个段落（块）：``[[ # ^ ]]``

为链接创建定义（关键词）：``[[ | 关键词]]``

链接到外部文件如印象笔记：``[关键词]（链接）``
